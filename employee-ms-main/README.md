# Spring Boot Employee Management System

A backend Employee Management System developed using Spring Boot, Java, and REST APIs. This project performs CRUD (Create, Read, Update, Delete) operations for managing employee records with MySQL database integration.

The application demonstrates backend development concepts including REST API implementation, layered architecture, Spring MVC, dependency injection, and database connectivity using Spring Data JPA.

## API Endpoints

| HTTP Method | Endpoint                             | Description       |
| ----------- | ------------------------------------ | ----------------- |
| GET         | /api/v1/employee/getAllEmployees     | Get all employees |
| POST        | /api/v1/employee/saveEmployee        | Add employee      |
| GET         | /api/v1/employee/searchEmployee/{id} | Search employee   |
| PUT         | /api/v1/employee/updateEmployee      | Update employee   |
| DELETE      | /api/v1/employee/deleteEmployee/{id} | Delete employee   |
